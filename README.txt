Corner2Speech

To run the software
    Open a Windows CMD shell and change directory to here
    Type "python Corner2Speech.py" and press <enter>

The following tracks are supported

    Circuit of the Americas - Grand Prix
    Hockenheimring Baden-Wurttemberg - Grand Prix
    Motorsport Arena Oschersleben - Grand Prix
    Nurburgring Combined - Gesamtstrecke 24h
    Nurburgring Combined - Gesamtstrecke VLN
    Nurburgring Grand-Prix-Strecke - Grand Prix w/out Arena
    Nurburgring Grand-Prix-Strecke - Grand Prix
    Nurburgring Nordschleife - Industriefahrten
    Nurburgring Nordschleife - Touristenfahrten
    Okayama International Circuit - Full Course
    Okayama International Circuit - Short
